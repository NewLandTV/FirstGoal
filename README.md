# First Goal

Goal Game in Discord.

## How to run

### Clone this repository

```sh
git clone https://github.com/NewLandTV/FirstGoal.git
cd FirstGoal
```

### Install packages

```sh
npm install
```

### Setting config.json

|Name|Description|
|-|-|
|clientId|Bot application id|
|guildId|Your guild id|
|token|Bot token|

### Run

```sh
deploy
run
```
